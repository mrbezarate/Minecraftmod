# Исправление загрузки плагинов Mineflayer

## Проблема
Плагины `mineflayer-pvp`, `mineflayer-collectblock` и `mineflayer-tool` не загружаются, так как их структура экспорта отличается от стандартной (функция находится в свойстве `.plugin`).

## Решение
Обновить логику загрузки плагинов в файле `api/bot/index.ts`.

## Шаги реализации
1.  **Рефакторинг функции `load` в `api/bot/index.ts`**:
    - Добавить проверку свойства `.plugin` у загружаемого объекта.
    - Приоритет проверки: `plugin.plugin` -> `plugin.default` -> `plugin`.
    - Это позволит корректно инициализировать все типы плагинов.

2.  **Проверка**:
    - Перезапуск сервера.
    - Мониторинг логов на наличие сообщений `Plugin ... loaded successfully`.
